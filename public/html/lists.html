Create an AI-generated List

How to Use

You can create a list using the following keywords:

  • new - To get the newest books
  • old - To get older books
  • less/more than xxx pages - To filter by page count
  • after/before xxxx years - To filter by publication year
  • include categories! - Specify genres like romance, fiction, mystery, etc.

Example queries:

  • new fiction books less than 300 pages
  • old romance books
  • mystery books before 2000
